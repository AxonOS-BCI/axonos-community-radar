This repository tracks public BCI, neurotechnology, real-time Rust, privacy, signal-processing and cognitive infrastructure projects relevant to AxonOS.

It does not auto-comment, auto-react or mass-follow.

The radar creates a review issue with:

- active repositories
- recent releases
- potential contribution targets
- manual outreach candidates

AxonOS engagement policy:

1. Star only relevant repositories.
2. React only to releases that are genuinely relevant.
3. Open PRs only when they add technical value.
4. Avoid low-signal comments.
5. Prefer tests, docs, reproducibility, benchmarks and protocol notes.

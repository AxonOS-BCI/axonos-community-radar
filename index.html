How this radar works

discovery, not endorsement

The radar is generated from public GitHub topic search. A repository is kept if it carries a core BCI/neuro topic, or a context topic (signal processing, embedded, privacy…) plus an anchored neuro keyword — so generic projects don't leak in. Recency is measured from a fixed 6-hour snapshot, so unchanged projects keep a stable position. It's a discovery map — not an endorsement or a ranking of quality, safety, or clinical fitness. AxonOS is ranked by the same formula as everyone else.

Evidence tiers

why each project qualifies
L3Explicit BCI. Carries a direct brain–computer-interface / neural-interface topic. The clearest signal of relevance.
L2Neural signal. A neural-signal topic or keyword (EEG, ECoG, EMG, spike sorting, motor imagery…) without an explicit BCI tag.
L1Context + keyword. A context topic (signal processing, embedded, real-time, privacy) plus an anchored neuro keyword in the metadata.
L0Weak adjacency. Included but flagged for review — a borderline match that may be a false positive.

Scoring

transparent, neutral

Position is a neutral discovery signal: log₁₀(stars+1) × 10 + recency. Rising reflects real 7-day star velocity from accumulated history. No hand-tuned “hype”, no paid placement, no self-boosting — every project, including AxonOS, is ranked by the same formula.

What we're building

the AxonOS engineering roadmap

AxonOS is an open, real-time neural operating system for brain–computer interfaces — a no_std Rust kernel, a wire-level consent protocol, a deterministic signal pipeline, and the standard that ties them together. This is what's shipped, in progress, and planned.

BetaKernel

axonos-kernel

Real-time no_std kernel: deterministic scheduler, SPSC queues, monotonic time, capability model.
ShippedConsent & Protocol

axonos-consent

Kernel-level consent FSM with formally-bounded withdrawal latency. Live MMP interop test PASS.
ShippedConsent & Protocol

axonos-protocol

The AxonOS Consent Protocol on the wire: bounded CBOR, exhaustive FSM, hardware StimGuard.
ShippedSignal & ML

axonos-signal-pipeline

Fixed-point DSP chain: raw frame to typed decision, vector-pinned and deterministic.
ShippedStandard

axonos-standard

The normative standard and the CLAIMS.md catalogue that keeps every claim verifiable.
BetaStandard

axonos-conformance

Byte-exact wire-format vectors and codecs across several languages.
BetaCommunity

axonos-community-radar

This radar: a living, momentum-first map of the open BCI field.
